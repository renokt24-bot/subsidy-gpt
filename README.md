# subsidy-gpt — 助成金マッチングツール

会社情報を入力すると、**使える可能性のある助成金・補助金がすぐにわかる**ブラウザツールです。
インストールやAPIキーは不要。`index.html` を開くだけで動きます。

## 使い方

1. このリポジトリを取得して `index.html` をブラウザで開く
   ```bash
   # もっとも簡単な方法（ダブルクリックでもOK）
   open index.html        # macOS
   # または簡易サーバーで
   python3 -m http.server 8000   # → http://localhost:8000
   ```
2. 会社情報を入力する
   - 所在地（都道府県）／業種／従業員数／資本金／創業からの年数
   - **やりたいこと**（採用、正社員化、研修、賃上げ、設備投資、IT導入、販路開拓、創業 など）を選択
3. 「使える助成金を調べる」を押すと、条件に合う制度が一覧表示されます
   - **あてはまる点** … なぜ候補になったか
   - **確認・準備が必要な条件** … 申請にあたって満たすべき要件
   - 公式サイトへのリンク

## 制度を追加・編集するには

すべての制度データは [`data.js`](./data.js) に定義されています。
`SUBSIDIES` 配列にオブジェクトを追加するだけで新しい制度を収録できます。

```js
{
  id: "example",
  name: "制度名",
  org: "所管",
  category: "カテゴリ",
  amount: "支給額の目安",
  summary: "概要",
  url: "https://...",
  match(c) {
    // c: { prefecture, industry, employees, capital, yearsInBusiness, purposes[] }
    if (!c.purposes.includes("equipment")) return { eligible: false };
    return {
      eligible: true,
      reasons: ["あてはまる理由"],
      cautions: ["確認が必要な条件"],
    };
  },
}
```

中小企業・小規模事業者の判定には `isSME(c)` / `isSmallBusiness(c)` を利用できます。

## ファイル構成

```

```

## 注意

本ツールの判定は **申請の目安** です。各制度の要件・支給額・募集状況は簡略化しています。
実際の支給可否は、必ず厚生労働省・中小企業庁・各自治体・お近くの労働局や商工会議所等の
**公式な最新情報** でご確認ください。
