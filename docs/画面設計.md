# 画面設計書 — 助成金マッチングツール（subsidy-gpt）

## 1. 画面一覧

| ID | 画面名 | 役割 |
| --- | --- | --- |
| S1 | 入力フォーム画面 | 会社情報を入力して検索する |
| S2 | 補助金一覧画面 | 条件に合う助成金・補助金を一覧表示する |

両画面は1つの `index.html` 内のセクションとして実装し、JavaScriptで表示を切り替える（SPA方式）。

## 2. 画面遷移図

```
        ┌──────────────────────┐
        │ S1 入力フォーム画面     │
        │                      │
        │ [使える助成金を調べる] ──┼──▶ 入力チェックOK
        └──────────────────────┘        │
                  ▲                      ▼
                  │            ┌──────────────────────┐
        [← 条件を変更] ────────┤ S2 補助金一覧画面       │
                               │  - 要件マッチ助成金      │
                               │  - 募集中の公式補助金    │
                               └──────────────────────┘
                                          │
                              [公式の詳細を見る →] ──▶ 別タブで公式サイト
```

- 入力チェックNG時はS1に留まり、エラーメッセージを表示。
- 「← 条件を変更」でS1へ戻る（入力値は保持）。

## 3. 共通要素

- **ヘッダー（hero）**: タイトル「助成金マッチング」＋サブコピー。全画面共通で上部に表示。
- **フッター（disclaimer）**: 判定は目安である旨の免責文。全画面共通。
- **配色**: ブランドカラー＝青（`#2563eb`）。公式補助金カードは水色アクセント（`#0ea5e9`）。
- **レスポンシブ**: 画面幅560px以下でフォームを1カラム化。

## 4. S1 入力フォーム画面

### ワイヤーフレーム

```
┌────────────────────────────────────────┐
│            助成金マッチング               │  ← hero
│  会社情報を入力すると…すぐにわかります       │
├────────────────────────────────────────┤
│  会社情報を入力                           │
│  ┌─────────────┐ ┌─────────────┐        │
│  │ 所在地(都道府県)│ │ 業種          │        │
│  └─────────────┘ └─────────────┘        │
│  ┌─────────────┐ ┌─────────────┐        │
│  │ 従業員数(人)  │ │ 資本金(万円)   │        │
│  └─────────────┘ └─────────────┘        │
│  ┌─────────────┐                        │
│  │ 創業からの年数 │                        │
│  └─────────────┘                        │
│  やりたいこと（複数選択可）                 │
│  [採用][正社員化][研修][賃上げ][設備投資]    │  ← チップ（チェック式）
│  [IT導入][販路開拓][創業][事業転換][両立支援] │
│                                          │
│  (エラー時) ⚠ すべての会社情報を入力…       │
│  ┌────────────────────────────────────┐ │
│  │        使える助成金を調べる            │ │  ← CTAボタン
│  └────────────────────────────────────┘ │
└────────────────────────────────────────┘
```

### 入力項目定義

| 項目 | コントロール | 必須 | 値 | バリデーション |
| --- | --- | --- | --- | --- |
| 所在地（都道府県） | セレクト | ○ | 47都道府県 | 未選択不可 |
| 業種 | セレクト | ○ | 製造業他／卸売／小売／サービス | 未選択不可 |
| 従業員数 | 数値入力 | ○ | 0以上の整数 | 未入力不可 |
| 資本金（万円） | 数値入力 | ○ | 0以上の整数 | 未入力不可 |
| 創業からの年数 | 数値入力 | ○ | 0以上の整数 | 未入力不可 |
| やりたいこと | チェックチップ | 任意 | 10種から複数 | なし |

### 「やりたいこと」の選択肢

採用・雇用 / 非正規の正社員化 / 社員研修・人材育成 / 賃上げ / 働き方改革・両立支援 / 設備投資・生産性向上 / ITツール・DX導入 / 販路開拓・新商品開発 / 創業・起業 / 新分野展開・事業転換

### 操作・挙動

- 「使える助成金を調べる」押下 → 必須チェック → OKならS2へ遷移しつつ判定実行。
- NGならエラー文を表示し、最初の未入力項目にフォーカス。

## 5. S2 補助金一覧画面

### ワイヤーフレーム

```
┌────────────────────────────────────────┐
│            助成金マッチング               │  ← hero（共通）
├────────────────────────────────────────┤
│ [← 条件を変更]   N件の候補が見つかりました   │  ← results-bar
│  要件マッチ X件／募集中の公式補助金 Y件      │  ← サマリー
│                                          │
│ ■ 要件にマッチした助成金（雇用・人材系ほか）  │  ← セクション見出し
│  ┌────────────────────────────────────┐ │
│  │ [カテゴリ]                           │ │
│  │ キャリアアップ助成金（正社員化コース）   │ │
│  │ 厚生労働省                           │ │
│  │ 概要文…                             │ │
│  │ 支給額の目安: 40〜80万円程度           │ │
│  │ ✓ あてはまる点                       │ │
│  │   ・非正規社員の正社員化を予定している    │ │
│  │ ！ 確認・準備が必要な条件              │ │
│  │   ・雇用保険の適用事業所であること       │ │
│  │ 公式の詳細を見る →                    │ │
│  └────────────────────────────────────┘ │
│                                          │
│ ■ 募集中の補助金（国・地方自治体／公式データ）│
│  出典: jGrants…（データ更新: 2026/06/17）  │  ← 出典・更新日
│  ┌────────────────────────────────────┐ │
│  │ [東京都] [300名以下]                  │ │  ← 地域・規模バッジ
│  │ 【○○市】令和8年度 ○○応援補助金       │ │
│  │ 補助上限: 約200万円                   │ │
│  │ 募集期間: 2026/06/01 〜 2026/12/25     │ │
│  │ jGrantsで詳細・申請する →              │ │
│  └────────────────────────────────────┘ │
│  ほか N件（締切が近い順に40件を表示）        │
└────────────────────────────────────────┘
```

### 表示構成

1. **ヘッダーバー**: 「← 条件を変更」ボタン ＋ 「N件の候補が見つかりました」
2. **サマリー**: 「要件マッチ X件／募集中の公式補助金 Y件」
3. **セクションA: 要件マッチ助成金**（ルールベース）
   - カード項目: カテゴリ／制度名／所管／概要／支給額目安／あてはまる点／確認が必要な条件／公式リンク
4. **セクションB: 募集中の補助金**（jGrants）
   - 出典・データ更新日を明記
   - カード項目: 地域バッジ／従業員規模バッジ／制度名／実施機関／補助上限／募集期間／jGrantsリンク
   - 締切が近い順・最大40件表示

### 状態（ステート）

| 状態 | 表示 |
| --- | --- |
| 候補あり | サマリー＋該当セクション・カード群 |
| 候補なし | 「条件に一致する制度はありませんでした」＋再検索の案内文 |

### カードのバリエーション

| 種類 | 左ボーダー色 | 主な情報 |
| --- | --- | --- |
| 要件マッチ助成金 | 青（標準） | あてはまる点 / 確認が必要な条件 |
| 公式補助金（jGrants） | 水色 | 地域・規模バッジ / 補助上限 / 募集期間 |

## 6. 文言・トーン

- 専門用語にはふりがな的な補足を添え、非エンジニア・非専門家にも分かる平易な日本語。
- 断定を避け「使える可能性」「目安」と表現。最終確認は公式情報へ誘導。
