# 全力ストレッチ 採用サイト

デザインコンセプト「静かな熱量 — 高級サロンの品格 × アスリートの本気」に基づく、1ページ完結のスクロール型採用LP + 応募フォーム。

## 構成

```
recruit/
├── index.html   採用トップ(ヒーロー/実績/代表メッセージ/仕事/先輩の声/キャリア/給与/FAQ/エントリー)
├── entry.html   応募フォーム(4項目・約30秒/?type=tour で見学モード)
├── css/
│   ├── style.css   共通スタイル(デザイントークンは:rootに集約)
│   └── entry.css   フォーム専用
└── js/
    ├── main.js     ローディング/スクロール出現/カウントアップ/パララックス/追従CTA
    └── entry.js    フォームバリデーション/見学モード切替
```

ビルド不要の静的サイト。GitHub Pages なら `https://<user>.github.io/<repo>/recruit/` で公開できます。
